This is the API

getCandidate

Returns a candidate by ID

URL

https://virtserver.swaggerhub.com/Face-the-Facts/ParliamentAPI/v1/candidates/{id}

HTTP Method

GET

Response Type

Candidate

Parameters

  • Path: id Integer

    The ID of the candidate to return

getCandidates

Returns a list of candidates

URL

https://virtserver.swaggerhub.com/Face-the-Facts/ParliamentAPI/v1/candidates

HTTP Method

GET

Response Type

List

Parameters

  • Query: limit Integer

    Limits the number of items on a page

  • Query: offset Integer

    Specifies the page number of the artists to be displayed

  • Query: party Integer

    ID of the party to be filtered by

postCandidate

Create a new Candidate

URL

https://virtserver.swaggerhub.com/Face-the-Facts/ParliamentAPI/v1/candidates

HTTP Method

POST

Response Type

inline_response_201

Parameters

  • Body: body Candidate(Candidate)

    Optional description in *Markdown*
